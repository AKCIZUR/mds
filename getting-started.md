# Rychlý start

## Instalace

```bash
pip install -r requirements.txt
```

## Lokální spuštění

```bash
mkdocs serve
```

## Produkční build

```bash
mkdocs build --strict
```

## Doporučený postup

1. uprav `site_url`
2. uprav `repo_url`
3. doplň logo a favicon
4. vyplň českou i anglickou strukturu
5. dolaď code highlighting podle technologií projektu
