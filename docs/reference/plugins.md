# Plugin stack

## Used stack

- `search` for built-in site search
- `minify` for output trimming
- `glightbox` for image lightbox zoom
- `literate-nav` for Markdown-based navigation
- `section-index` for clickable section pages

## Why this stack works

- the search plugin is the built-in default in MkDocs
- Material for MkDocs supports navigation, content tabs, and code presentation features
- the additional plugins are established, documented, and widely used
- the CSS theme layer carries the shadcn look without introducing unstable dependencies
