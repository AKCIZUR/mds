# Rychlý start

## Instalace

```bash
pip install -r requirements.txt
python scripts/generate_indexes.py
mkdocs serve
```

## Build

```bash
mkdocs build --strict
```

## Doporučený postup

1. upravit `site_url` a `repo_url`
2. doplnit vlastní obsah
3. spustit generování indexu
4. nasadit přes GitHub Actions
