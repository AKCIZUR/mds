# Getting started

<div data-lang="cs">
Tato šablona je připravená pro GitHub Pages. Po instalaci závislostí spusťte `mkdocs serve` a upravte si texty, barvy a navigaci.
</div>

<div data-lang="en">
This template is ready for GitHub Pages. After installing dependencies, run `mkdocs serve` and adapt the text, colors, and navigation.
</div>

## Fast setup

```bash
pip install -r requirements.txt
mkdocs serve
```

## Notes

- keep the docs small and readable
- use reusable blocks for callouts and code
- let the skeleton handle the first impression
- shadcn styling lives in `docs/assets/custom.css`
