# VCCSD Docs Pro

Production documentation site with search, versions, blog, API reference and two languages.

## Ready out of the box

- full-text search
- CZ / EN structure
- automatic menu
- GitHub edit links
- command palette `Ctrl+K`
- lazy loading
- skeleton loading
- blog
- versioned docs
- Mermaid
- OpenAPI

## Quick entry

```bash
mkdocs serve
```
