# FAQ

## Is it light/dark switchable?

The base stays dark-first, but the new layer makes the UI softer, cleaner, and more app-like.

## Is it ready for GitHub Pages?

Yes. The workflow is part of the repository.

## Can I use it as a base for another project?

Yes. Just replace branding, URLs, and content.

## What should be adjusted first?

- `USERNAME`
- `REPO`
- logo
- favicon
- Czech and English content
