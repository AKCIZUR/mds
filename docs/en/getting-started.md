# Getting started

## Install

```bash
pip install -r requirements.txt
```

## Run locally

```bash
mkdocs serve
```

## Production build

```bash
mkdocs build --strict
```

## Recommended flow

1. update `site_url`
2. update `repo_url`
3. add your logo and favicon
4. fill both Czech and English content
5. refine code highlighting for your stack
