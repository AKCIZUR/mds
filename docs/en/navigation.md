# Navigation and layout

This starter is designed for documentation that should feel fast, modern, and easy to scan.

## Recommended menu structure

- Home
- Start
- Menu and layout
- Reference
- FAQ
- Release notes
- Changelog
- English mirror

## Panel layout

| Panel | Purpose |
| --- | --- |
| Left | documentation tree |
| Top | identity, search, links |
| Right | TOC / page outline |
| Center | the article itself |

## Note

!!! tip
    A compact and logical menu makes the whole site feel significantly more premium.
