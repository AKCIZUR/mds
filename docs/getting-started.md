# Rychlý start

## Instalace

```bash
pip install -r requirements.txt
```

## Lokální spuštění

```bash
mkdocs serve
```

## Produkční build

```bash
mkdocs build --strict
```

## Co upravit hned

1. `site_url`
2. `repo_url`
3. `repo_name`
4. logo a favicon v `docs/assets/`
5. vlastní obsah v `docs/`

!!! note
    Starter je schválně jednoduchý. To je výhoda. Minimalismus bez dír v lodi.
