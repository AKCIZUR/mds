# Navigace a layout

Tento základ je navržený pro dokumentaci, která má působit rychle, moderně a čitelně.

## Doporučené členění menu

- Domů
- Start
- Menu a layout
- Reference
- FAQ
- Release notes
- Changelog
- English mirror

## Panelové rozvržení

| Panel | Účel |
| --- | --- |
| Levý | strom dokumentace |
| Horní | identita, search, odkazy |
| Pravý | TOC / obsah stránky |
| Střed | samotný článek |

!!! tip
